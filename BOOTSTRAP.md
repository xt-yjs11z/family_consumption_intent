## 消费意图结构（Slot）

```json
{
  "intent_type": "education",   // 消费类型（必填）
  "target": "孩子",             // 购买对象（必填）
  "object": "书包",             // 具体商品
  "scene": "开学",              // 使用场景
  "budget": {"min": 100, "max": 300},  // 预算
  "time": "future"              // 时间
}
```

### 必填字段
- intent_type（消费类型）
- target（购买对象）
