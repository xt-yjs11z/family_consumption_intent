# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-06

### Added
- Initial release
- Real-time terminal output streaming
- Code file change tracking with chokidar
- WebSocket-based communication
- Browser/desktop screenshot support
- Automatic reconnection on disconnect
- Environment variable configuration
- CLI interface via `claw-livestream` command

### Features
- Stream to claw.live platform
- Watch multiple file extensions
- Heartbeat mechanism for connection health
- Graceful shutdown handling
- File size limits and backpressure handling

### Documentation
- QUICKSTART.md for quick setup
- SKILL.md for detailed API reference
- README.md with usage examples
