# 2026-03-12 工作日志

## 家庭消费意图识别系统 (family-intent-recognition)

### 主要功能
- 意图识别：识别消费意图的类别、商品、消费对象、场景
- LLM对话：多轮对话追问补全信息
- 家庭画像：基于用户画像的个性化建议

### 字段结构
- intent: 是否有消费意图
- category: 消费品类（电子产品、家电、服装、食品、娱乐等）
- target: 具体商品
- target_object: 消费对象（孩子、父母、配偶、自己、家庭）
- scenario: 消费场景（节假日、教育、医疗等）
- follow_up_question: 追问问题
- pending_fields: 待追问字段队列
- spending_advice: 消费建议

### 多轮对话追问逻辑
1. 商品未知时追问具体商品
2. 消费对象未知时追问给谁买
3. 回答后自动更新字段并继续追问
4. 商品确定后自动更新类别

### 优化记录
- V1.0.2: 基础版本
- V1.0.3+: 优化意图识别字段、消费对象、追问逻辑
